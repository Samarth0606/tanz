## This is an Ecommerce Application

### Key fetures :

- **CRUD operation on Product**
- **Review on Product**
- **Likes on Product**
- **Authentication**
- **Authorisation**
- **Add to cart**
- **Payment gateway**

# Technology used:

- ejs
- nodemon
- express
- mongoose
- express-session
- cookie-parser
- passport
- passport-local
- passport-local-mongoose
- joi
- method-override
- conect-flash

### Steps to follow for development stage

We have 1 main folder:
`Ecommerce-tanz`

- Change your directory to `Ecommerce-tanz`
- npm install
- npm start
  `(make sure you .env file has the SECRETS)`
